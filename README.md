# SOC-Beginner-Guide
Welcome! This is a free guide for anyone who wants to learn about Security Operations Centers (SOC).

## 📚 What is a SOC?
A SOC (Security Operations Center) is a team that monitors and protects 
an organization from cyber attacks 24/7.

## 🗺️ Beginner Roadmap

1. ✅ Learn what a SOC is
2. ✅ Understand SIEM (Security Info & Event Management)
3. ✅ Try Wazuh (free SIEM tool)
4. ✅ Learn about TheHive (incident response)
5. ✅ Practice threat hunting
6. ✅ Study MITRE ATT&CK framework
